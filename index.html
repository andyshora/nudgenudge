nugenudge

jQuery plugin to nudge your users towards goals with annoying arrows, by @andyshora

Notes

  • Arrows point via CSS3 animations.
  • Nudge length is 30-50px horizontally, regardless of arrow size. Sorry, I haven't had time to create dynamic keyframe animations yet, but feel free to edit the CSS!

Supported Browsers

  • Supported browsers: Chrome, Firefox, Safari, probably IE10 if they've got their act together.

Demo & Download

Do your users need nudging in this direction? Then apply the nudgenudge jQuery plugin!

Usage

Files required: js/jquery.nudgenudge.x.x.js, css/nudgenudge.css, img/your_arrow_image.png

Initialize the plugin when the document ready event has fired:

$('#signup').nudgenudge({
	arrow: 'img/arrow_default.png', // path to the arrow image
	arrowWidth: 150,
	arrowHeight: 60,
	intensity: 'medium',  // the intensity of the nudge (low, medium, high)
	placement: 'left', // place on the left or the right of the target
	closeEvent: { "el": '#signup', "event": 'click' }, // selector and event which triggers arrow hiding
	hideAfter: 0,  // hide after this many nudges, 0 = for the rest of eternity
	offsetX: -5, // adjust x position
	offsetY: 0 // adjust y position
});
